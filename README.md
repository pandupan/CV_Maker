# 📄 CV Maker — Koleksi Template CV Modern

Kumpulan **9 template CV (Curriculum Vitae)** yang dikonversi dari desain gambar menjadi kode HTML + Tailwind CSS. Setiap template dioptimalkan untuk lebar 800px, siap pakai untuk keperluan lamaran kerja.

## ✨ Fitur

- **9 Template Unik** — Berbagai desain CV dari minimalis hingga kreatif
- **HTML + Tailwind** — Kode bersih, mudah dimodifikasi dan dikustomisasi
- **800px Optimized** — Didesain untuk lebar optimal di layar desktop
- **Siap Cetak** — Bisa langsung di-print atau disimpan sebagai PDF
