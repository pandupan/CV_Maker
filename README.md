# 📄 CV Maker

Kumpulan template **CV (Curriculum Vitae)** modern yang dikonversi dari desain gambar ke HTML + Tailwind CSS dengan lebar 800px.

## ✨ Fitur

- **9 Template CV** — Berbagai desain CV yang siap digunakan
- **HTML + Tailwind** — Kode bersih dan mudah dimodifikasi
- **Responsif** — Tampilan optimal di layar 800px

## 🛠️ Tech Stack

- **HTML5** — Struktur halaman
- **Tailwind CSS** — Utility-first styling (inline)

## 🚀 Cara Melihat

Buka salah satu file `CV_*.html` langsung di browser.

```
CV_1.html  CV_2.html  CV_3.html  ...  CV_9.html
```
